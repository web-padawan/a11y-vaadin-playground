# A11y playground

This is a project for testing Vaadin web components accessibility improvements.

## Setup

```sh
npm install
```

## Run dev server

```sh
npm start
```

## Components

The following components are included:

- `vaadin-button`
- `vaadin-email-field`
- `vaadin-number-field`
- `vaadin-password-field`
- `vaadin-text-area`
- `vaadin-upload`
